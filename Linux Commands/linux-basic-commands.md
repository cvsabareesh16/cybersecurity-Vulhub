# Linux Basic Commands for Cybersecurity

## Navigation

```bash
pwd
ls
ls -la
cd foldername
cd ..
```

## File Reading

```bash
cat file.txt
less file.txt
head file.txt
tail file.txt
```

## User Information

```bash
whoami
id
hostname
uname -a
```

## Network Commands

```bash
ip a
ifconfig
ping target_ip
netstat -tulnp
ss -tulnp
```

## Searching

```bash
find / -name filename 2>/dev/null
grep -r "password" /path 2>/dev/null
```
